install blog

手把手教你从0开始搭建自己的个人博客 |无坑版视频教程| hexo

安装node.js

安装git

安装git部署插件
npm install –save hexo-deployer-git

新建博客文件夹
mkdir blog

新建博客文章
hexo n “my-new-blog”

hexo清理
hexo clean

hexo重新生成
hexo g

hexo预览
hexo s

hexo部署到github(_config.yml添加git配置)
hexo d
